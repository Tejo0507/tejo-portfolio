## Where to Add Your Profile Picture

Place your profile picture in this folder (`images/`) with the filename: **`profile.jpg`**

## Image Requirements

### Size:
- **Recommended**: 500x500 pixels (square format)
- **Minimum**: 300x300 pixels
- **Maximum**: 1000x1000 pixels (will be automatically resized)

### Format:
- **Preferred**: JPG or JPEG
- **Also supported**: PNG
- **File size**: Try to keep under 500KB for fast loading

## How to Get/Create Your Profile Picture

### Option 1: Use an Existing Photo
- Find a good photo of yourself
- Crop it to a square (1:1 ratio)
- Resize to 500x500px
- Save as `profile.jpg`

### Option 2: Take a New Photo
- Use your phone or camera
- Good lighting (natural light works best)
- Clean background
- Look at the camera
- Take multiple shots and pick the best one

### Option 3: Professional Headshot
- If you have a professional headshot, that's perfect!
- Crop and resize as needed

## Quick Steps:

1. **Crop to Square**
   - Make sure it's 1:1 ratio (square)
   - Center your face in the frame

2. **Resize**
   - Set dimensions to 500x500 pixels
   - Maintain aspect ratio

3. **Optimize**
   - Compress if over 500KB
   - Use JPG format for photos

4. **Rename**
   - Save as: `profile.jpg` (lowercase, no spaces)

5. **Place in Folder**
   - Put it in the `images` folder
   - Replace any existing profile.jpg

## Testing

After adding your photo:
1. Open `index.html` in your browser
2. Your photo should appear on the left page (profile page)
3. It should be circular with a red border
4. If it doesn't show, check:
   - Filename is exactly `profile.jpg`
   - File is in the `images` folder
   - Clear browser cache (Ctrl+F5)

## Temporary Solution

If you don't have a photo ready right now:
- The portfolio will still work
- You'll see a broken image placeholder
- You can add the photo later without any issues
